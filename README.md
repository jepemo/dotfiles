# dotfiles
My Arch/Ubuntu dotfiles

## Installation

For install or update

```bash
# First Installation
git clone https://github.com/jepemo/dotfiles

# update
cd dotfiles
git pull origin master

# Install/Update
# For ubuntu
./install-or-update.sh ubuntu

# For arch
./install-or-update.sh arch
```
